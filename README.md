**Цель проекта:**
- создать python-скрипт, который получает из консоли команду (python main.py --files file.log --date 2025-07-22 --report average), и создает отчет в консоли в виде красивой таблицы.

**Стек**:
- Python 3.12.4
- tabulate
- pytest

**Структура проекта**:
- main.py - скрипт
- tests.py - тесты
